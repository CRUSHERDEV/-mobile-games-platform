## ğŸ¯ Quick Start

### Option 1: Fully Automated (Recommended)

**Set up once, posts daily forever:**

1. **Clone the repository**
   ```bash
   git clone https://github.com/CRUSHERDEV/-mobile-games-platform.git
   cd -mobile-games-platform
   npm install
   ```

2. **Get API Keys** (15 minutes)
   - News API: https://newsapi.org/register (FREE)
   - OpenAI: https://platform.openai.com/api-keys (~$0.10/month)

3. **Add to GitHub Secrets**
   - Go to: Settings â†’ Secrets â†’ Actions
   - Add `NEWS_API_KEY`
   - Add `OPENAI_API_KEY`

4. **Enable GitHub Actions**
   - Go to Actions tab
   - Enable workflows
   - Done! Posts publish daily at 9 AM UTC

5. **Deploy to Vercel**
   - https://vercel.com/new
   - Import your GitHub repo
   - One-click deploy

**See**: [FULL_AUTOMATION_GUIDE.md](./FULL_AUTOMATION_GUIDE.md) for detailed instructions
